เชื่อมโปรแกรมราคายางเข้ากับ Google Drive

คู่มือจับมือทำ 2 ส่วน — โฮสต์ไฟล์ให้เปิดได้ทุกเครื่อง แล้วเปิดสิทธิ์ Google Drive ใช้เวลารวมประมาณ 20–30 นาที ทำครั้งเดียวจบ

⏱️ ~25 นาที 💸 ฟรีทั้งหมด 🔁 ทำครั้งเดียว

ทำไมต้องมี 2 ส่วนนี้ก่อน?

Google บังคับว่าหน้าเว็บที่จะต่อ Drive ได้ ต้องเปิดผ่าน https:// (ไม่ใช่ดับเบิลคลิกไฟล์) — และการที่ หลายเครื่องเห็นข้อมูลตรงกัน ก็ต้องมีที่อยู่ (URL) กลางอยู่แล้ว ดังนั้น ส่วนที่ 1 (โฮสต์) จึงเป็นรากฐานที่เลี่ยงไม่ได้ ส่วน ส่วนที่ 2 (สิทธิ์ Drive) คือกุญแจให้โปรแกรมอ่าน-เขียนไฟล์ใน Drive ของคุณได้

1

โฮสต์ไฟล์บน GitHub Pages

เอาโปรแกรมขึ้นเว็บฟรี เพื่อให้ได้ลิงก์ https:// ที่เปิดได้ทุกเครื่อง

สมัคร GitHub (ถ้ายังไม่มี)

เข้า github.com/signup สมัครด้วยอีเมล ฟรี — เป็นที่เก็บโค้ด/เว็บที่ใช้กันทั่วโลก

สร้างที่เก็บไฟล์ (Repository) ใหม่

กดปุ่ม + มุมขวาบน → New repository

  • ช่อง Repository name ตั้งชื่อสั้นๆ เป็นภาษาอังกฤษ เช่น tire-price
  • เลือก Public (จำเป็น — Pages ฟรีต้องเป็นสาธารณะ · แต่ไม่ต้องห่วง ไฟล์ข้อมูลราคาจริงอยู่บน Drive ส่วนตัว ไม่ได้อยู่ในนี้)
  • กด Create repository

เปลี่ยนชื่อไฟล์หลักเป็น index.html

ก่อนอัปโหลด ให้เปลี่ยนชื่อไฟล์ โปรแกรมแก้ราคายาง Excel v2.html เป็น index.html — เพื่อให้ลิงก์สั้นสวย เปิดได้เลยไม่ต้องต่อท้ายชื่อไฟล์ยาวๆ

💡
ผมเปลี่ยนชื่อให้ได้เลยถ้าต้องการ — บอกมาได้ทันที

อัปโหลดไฟล์โปรแกรมทั้งหมด

ในหน้า repo กด Add file → Upload files แล้ว ลากทั้งโฟลเดอร์โปรแกรม เข้าไป ต้องมีครบ:

  • index.html (ไฟล์หลักที่เปลี่ยนชื่อแล้ว)
  • โฟลเดอร์ app-xls2/ ทั้งโฟลเดอร์ (css + js ทั้งหมด)
  • โฟลเดอร์ app-xls/ (ไฟล์ข้อมูลตั้งต้น)

เลื่อนลงล่าง กด Commit changes

⚠️
ต้องอัปโฟลเดอร์ app-xls2/ และ app-xls/ ขึ้นไปด้วย ไม่ใช่แค่ไฟล์ html อย่างเดียว ไม่งั้นจะหน้าขาว

เปิดใช้งาน GitHub Pages

  • ในหน้า repo กดแท็บ Settings
  • เมนูซ้าย กด Pages
  • หัวข้อ Branch เลือก main และโฟลเดอร์ / (root) แล้วกด Save

รอสัก 1–2 นาที รีเฟรชหน้า จะได้ลิงก์เว็บของคุณ หน้าตาแบบนี้:

🔗 ลิงก์เว็บของคุณ (URL)
https://ชื่อบัญชี.github.io/tire-price/
✅
ลองเปิดลิงก์นี้ในมือถือ/อีกเครื่อง — ถ้าโปรแกรมขึ้นปกติ = ส่วนที่ 1 เสร็จ! เก็บลิงก์นี้ไว้ ใช้ต่อในส่วนที่ 2
2

เปิดสิทธิ์ Google Drive (OAuth)

สร้าง “กุญแจ” ให้โปรแกรมอ่าน-เขียนไฟล์ใน Drive ของคุณได้อย่างปลอดภัย

เข้า Google Cloud Console

เปิด console.cloud.google.com ล็อกอินด้วยบัญชี Google เดียวกับ Drive ที่อยากให้เก็บไฟล์

สร้างโปรเจกต์ใหม่

มุมซ้ายบน กดเมนูเลือกโปรเจกต์ (ข้างคำว่า Google Cloud) → New Project → ตั้งชื่อ เช่น TirePrice → Create → รอสักครู่แล้วเลือกโปรเจกต์นี้

เปิดใช้งาน Google Drive API

  • เมนูซ้าย (☰) → APIs & Services → Library
  • ค้นคำว่า Google Drive API → กดเข้าไป → กด Enable

ตั้งค่าหน้าจอขออนุญาต (OAuth consent screen)

  • เมนูซ้าย → APIs & Services → OAuth consent screen
  • เลือกประเภท External → Create
  • กรอกชื่อแอป เช่น โปรแกรมราคายาง · ใส่อีเมลของคุณในช่อง support email และ developer email · ที่เหลือข้ามได้ กด Save and Continue ไปจนจบ
  • หน้า Test users → Add users → ใส่ อีเมล Google ของคุณ (และเครื่อง/คนที่จะใช้) → Save
💡
ใส่อีเมลตัวเองใน Test users ก็พอ ใช้งานได้เลยไม่ต้องส่ง Google ตรวจ (เหมาะกับใช้ส่วนตัว/ในร้าน)

สร้าง OAuth Client ID

  • เมนูซ้าย → APIs & Services → Credentials
  • กด + Create Credentials → OAuth client ID
  • ช่อง Application type เลือก Web application
  • หัวข้อ Authorized JavaScript origins → + Add URI → ใส่ เฉพาะส่วนต้นของลิงก์ (ไม่เอาชื่อ repo/ท้ายสุด):
ใส่ในช่อง Authorized JavaScript origins
https://ชื่อบัญชี.github.io
⚠️
สำคัญ: ช่องนี้ใส่แค่ https://ชื่อบัญชี.github.io เท่านั้น ห้ามมี /tire-price/ ต่อท้าย และห้ามมี / ปิดท้าย

กด Create

คัดลอก Client ID

จะมีหน้าต่างเด้งขึ้นมาพร้อม Your Client ID หน้าตาแบบนี้ — กดคัดลอกเก็บไว้:

🔑 Client ID (หน้าตาประมาณนี้)
8124xxxxxxxx-abcd1234efgh5678.apps.googleusercontent.com
🔑
นี่คือสิ่งเดียวที่ผมต้องการ — ส่ง Client ID นี้กลับมาให้ผม แล้วผมจะเขียนระบบซิงค์ Drive + เทสต์ให้จบทันที (Client ID ไม่ใช่รหัสลับ ใช้ฝั่งหน้าเว็บได้ ปลอดภัย)

✅ เสร็จแล้ว — ส่ง 2 อย่างนี้กลับมาให้ผม

พอได้ครบ ผมจะเขียน Google Drive sync ต่อให้จบทีละเฟส (เชื่อม → ซิงค์อัตโนมัติ → เก็บประวัติขึ้น cloud)

1) ลิงก์เว็บ  →  https://ชื่อบัญชี.github.io/tire-price/

2) Client ID  →  xxxx-xxxx.apps.googleusercontent.com

ติดตรงไหน บอกได้ทุกขั้น

ถ้าทำขั้นไหนแล้วงงหรือหน้าจอไม่เหมือนในคู่มือ แคปหน้าจอส่งมา ผมช่วยดูให้ทีละจุด — หรือถ้าอยากให้ผมเปลี่ยนชื่อไฟล์หลักเป็น index.html ให้ก่อน ก็บอกได้เลย
